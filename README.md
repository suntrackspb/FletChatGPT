## Installation docker
You can specify all the settings yourself in .env or fill them out in the application settings as a client. The client settings override the server settings for the client.
1. Clone repository
2. ```shell 
   cd FletChatGPT
   ```
3. ```shell 
   docker compose up -d
   ```
   
4. Open http://your_ip:8009
