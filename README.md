# Flet Kandinsky and ChatGpt PWA

## Run local
1. ```shell 
   git clone https://github.com/suntrackspb/FletChatGPT.git
   ```
2. ```shell 
   cd FletChatGPT
   ```
3. ```shell 
   source venv\bin\activate
   ```
4. ```shell 
   pip install -r requirements.txt
   ```
5. ```shell 
   python main.py
   ```
6. Open http://your_ip:8009
